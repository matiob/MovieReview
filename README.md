# # TMDB MovierReview

### Objective

Application that shows movies and TV shows. Consumes information from [_The Movie Database_ (TMDB)](https://www.themoviedb.org/) API.

### Features

- Search and list movies.
- View the details of a movie or TV show.
- Create users.
- _Login_ and _logout_ users.
- Add a movie or show to a favorites list.
- View a list of favorites.
- Remove a movie or show from a favorites list.
- Differentiate the _front-end_ paths for movies and TV shows.
- Search users.
- View the profile of a specific user (with their favorite movies or shows).
- Keep session open before closing the _browser_ or `refresh`.
